# Portfolio

Personal portfolio website built with React and Vite.

Live site: [cburke.netlify.app](https://cburke.netlify.app)

## Tech Stack

- React 19
- Vite 7
- React Router DOM
- Tailwind CSS 4
- Heroicons
- tech-stack-icons
- ESLint
- Netlify (deployment)

## How To Run

### 1. Install dependencies

```bash
npm install
```

### 2. Start development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build locally

```bash
npm run preview
```

### 5. Run linting

```bash
npm run lint
```
